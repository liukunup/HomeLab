# 开发环境

## 一键启动

```shell
curl -fsSL https://raw.githubusercontent.com/your-username/your-repo/main/deploy.sh | sh
```

```shell
curl -fsSL https://raw.githubusercontent.com/your-username/your-repo/main/deploy.sh -o deploy.sh
chmod +x deploy.sh
./deploy.sh
```
